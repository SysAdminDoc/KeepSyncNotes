# KeepSyncNotes

![License](https://img.shields.io/badge/license-MIT-blue)
![Language](https://img.shields.io/badge/language-Python-3776AB)
![Type](https://img.shields.io/badge/type-Desktop%20App-brightgreen)

A Google Keep importer and note tracking application. Import your Google Keep exports, organize notes, and manage them locally with a dark-themed desktop interface.

## Features

- **Google Keep Import** — Import notes from Google Takeout JSON exports
- **Note Management** — Create, edit, search, and organize notes locally
- **Tag System** — Categorize notes with labels and tags
- **Search** — Full-text search across all imported notes
- **Dark Theme** — Professional dark-themed interface
- **Local Storage** — All data stays on your machine

## Installation

```bash
python keepsync_notes.py
```

Dependencies auto-install on first run.

## Requirements

- Python 3.8+
